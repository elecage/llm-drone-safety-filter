# eval/

벤치마크·결함 주입·지표. 모든 정량 결과는 여기서 재현 가능해야 함.

## Layout

- `regression/` — 회귀 테스트. 안전 불변량 검증(e.g., 임의 setpoint → 버블 침입 0). PR/CI 후보.
- `faults/` — 결함 주입 하니스. 환각(좌표/대상 객체 교란), 적대 계획, 토이 프롬프트 주입 시나리오 생성기.
- `baselines/` — B0(필터 없음), B1(정적 고정마진), B2(신뢰도 변조). B1 vs B2 비교가 C2(신뢰도-변조 CBF) 입증 핵심.
- `metrics/` — 지표 산출: 안전위반율, 작업성공률, 개입/확인율, 과보수성, 레이턴시 예산 (결정론 루프 ≥20Hz 유지 검증), agency 지표 (사용자 주도성).

자세한 실험 설계는 [RESEARCH_CONTEXT §B6](../docs/RESEARCH_CONTEXT.md#b6-1차-논문-계획) "실험" 항목.
